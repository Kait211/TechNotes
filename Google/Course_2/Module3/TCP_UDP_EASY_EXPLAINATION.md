Here are simple definitions with examples:

* **TCP (Transmission Control Protocol)**: Like sending a package with tracking. It makes sure **all the data arrives, in the right order**. If something is missing, it sends it again.

  * **Used for:** 🌐 Websites, 📧 Email, 📁 File downloads, 🏦 Online banking
  * **Example:** When you download a game from Steam, TCP makes sure every file arrives correctly.

* **UDP (User Datagram Protocol)**: Like throwing paper airplanes. It **sends data as fast as possible** without checking if it arrives.

  * **Used for:** 🎮 Online games, 📹 Video calls, 🎥 Live streaming, 🎤 Voice chat
  * **Example:** In an online game, if one player's position update is lost, the game just sends the next update instead of waiting.

### Easy way to remember:

* 📦 **TCP = "Did you get it?"** → Reliable but slower.
* ✈️ **UDP = "Here it comes!"** → Fast but doesn't check if it arrived.
