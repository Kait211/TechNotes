### Application layer
Allows these applications to communicate in a way they understand 

# OSI Model

![OSI Model Diagram](../../images/OSI_Model.png.png)

### Session layer
Responsible for facilitating the communication between actual applications and the transport layer. Takes the application layer data and hands it off to the presentation layer 

### Presentation Layer
Responsible for making sure that the unencapsulated application layer data is able to be understood by the application in question 

- We believe a five-layer model is the most useful when ​it comes to the day-to-day ​business of understanding networking, ​but the seven-layer OSI model is also prevalent
