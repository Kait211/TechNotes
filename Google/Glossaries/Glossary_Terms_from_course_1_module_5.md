# Software Terms

- **Application Software:** Any software created to fulfill a specific need, like a text editor, web browser, or graphics editor.
- **Assembly Language:** A language that allowed computer scientists to use human-readable instructions, assembled into code that machines could understand.
- **Automation:** Makes processes work automatically.
- **Coding:** Translating one language into another.
- **Compiled Programming Language:** A language that uses human-readable instructions, then sends them through a compiler.
- **Copyright:** Used to protect original work.
- **.exe:** A file extension in Windows for an executable file.
- **Firmware:** Software that's permanently stored on a computer component.
- **Git:** A version control system that helps keep track of changes made to files and directories.
- **Interpreted Programming Language:** A language that isn't compiled ahead of time.
- **Programming:** Coding in a programming language.
- **Programming Language:** A special language software developers use to write instructions for computers to execute.
- **Script:** Code that is run by an interpreter, which translates it into CPU instructions just in time for execution.
- **Scripting:** Coding in a scripting language.
- **Software:** The intangible instructions that tell hardware what to do.
- **Software Bug:** An error in software that causes unexpected results.
- **Software Management:** A broad term for software designed to manage or help manage projects or tasks.
- **System Software:** Software used to keep the core system running, such as operating system tools and utilities.
