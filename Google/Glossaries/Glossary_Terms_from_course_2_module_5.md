# Glossary

**Ad-Hoc Network:** A network without infrastructure where devices communicate directly.

**Asymmetric Digital Subscriber Line (ADSL):** DSL connection with faster downloads than uploads.

**Baud Rate:** Number of bits sent per second over a connection.

**Bluetooth:** Short-range wireless network technology.

**Broadband:** Any Internet connection faster than dial-up.

**Cable Modem:** Connects a home network to a cable provider.

**Cable Modem Termination System:** Connects cable users to an ISP network.

**Channels:** Smaller sections of a wireless frequency band.

**Collision Domain:** Network area where only one device can send at a time.

**Data Payload Section:** Contains the actual data being transmitted.

**Dial-up:** Internet connection using phone lines.

**DSL:** Internet technology using phone lines for higher speeds.

**DSLAM:** Connects many DSL lines to a high-speed network.

**Duration Field:** Shows how long a frame lasts.

**Frame Check Sequence (FCS):** Checks if a frame has errors.

**Frame Control Field:** Contains information about how a frame is processed.

**Frequency Band:** A section of radio frequencies used for communication.

**FTTB (Fiber to the Building):** Fiber connection to a building.

**FTTH (Fiber to the Home):** Fiber connection directly to homes.

**FTTN (Fiber to the Neighborhood):** Fiber connection to a neighborhood cabinet.

**FTTP (Fiber to the Premises):** Fiber connection to a location/building.

**FTTX:** Fiber to a specific location (X).

**HDSL:** High-speed DSL technology.

**MAC Filtering:** Allows only approved MAC addresses to connect.

**Mesh Networks:** Devices connect and pass data between each other.

**Metered Connection:** Internet where usage is tracked or limited.

**Non-Metered Connection:** Unlimited Internet usage.

**Optical Network Terminator:** Converts fiber signals to copper network signals.

**Pairing:** Connecting wireless devices so they remember each other.

**Point-to-Point VPN:** Secure VPN tunnel between two locations.

**Receiving Address:** MAC address of the device receiving a frame.

**SDSL:** DSL with equal upload and download speeds.

**Sequence Control Field:** Keeps frames in the correct order.

**Short-Range Wireless Network:** Wireless connection for nearby devices.

**T-Carrier Technologies:** Systems that transmit large amounts of data over lines.

**Transmitter Address:** MAC address of the device sending a frame.

**WPA (Wi-Fi Protected Access):** Wireless security using encryption.

**Wide Area Network (WAN):** Network covering large geographic areas.

**WEP (Wired Equivalent Privacy):** Older, weak wireless encryption.

**Wireless Access Point:** Connects wireless devices to a wired network.

**Wireless LAN (WLAN):** Wireless network using access points.

**Wireless Networking:** Networks using radio signals instead of cables.
