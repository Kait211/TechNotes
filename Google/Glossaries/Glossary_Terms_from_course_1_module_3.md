# Glossary

* **Android**: A mobile operating system based on Linux.
* **Application**: A computer program designed for a specific use.
* **BIOS/UEFI**: Low-level software that initializes hardware during startup.
* **Block Storage**: Stores data in blocks for faster access and handling.
* **Boot**: The process of starting a computer.
* **Bootloader**: A small program that loads the operating system.
* **Chrome OS**: A Linux-based operating system developed by Google.
* **Command Line Interface (CLI)**: A text-based interface used to interact with the operating system.
* **Computer File**: A unit of stored data, such as a document, image, or song.
* **Data Blocks**: Pieces of data stored in different locations on a disk.
* **Distributions**: Variations of Linux, such as Ubuntu, Debian, and Red Hat.
* **Etcher.io**: A tool for creating bootable USB drives from operating system images.
* **File Extension**: The suffix of a filename that indicates its file type.
* **File Handling**: The process of storing and managing data with a program.
* **File System**: A method used to organize and manage files on storage devices.
* **Finder**: The default file manager for macOS.
* **Folders/Directories**: Containers used to organize files.
* **Hardware Resource Deficiency**: A shortage of resources such as memory or storage.
* **HFS+/APFS**: Apple's file systems used for storing and organizing data.
* **Hostname**: A name used to identify a computer on a network.
* **Input/Output (I/O) Device**: Hardware that sends or receives data, such as keyboards and monitors.
* **Install Image**: A file containing an operating system used for installation.
* **I/O Management**: The handling of data input and output operations.
* **iOS**: Apple's mobile operating system.
* **Kernel**: The core part of an operating system that manages hardware and processes.
* **Logs**: Files that record system events and activities.
* **macOS**: Apple's desktop operating system.
* **Memory Management**: The process of allocating and optimizing memory usage.
* **Metadata**: Information about a file, such as its creator and modification date.
* **Microsoft Terminal Services Client**: A program used to establish RDP connections.
* **OpenSSH**: A widely used suite of tools for secure remote access using SSH.
* **Operating System (OS)**: Software that manages hardware resources and provides services to applications.
* **PC (Personal Computer)**: A computer designed for individual use.
* **Plink (PuTTY Link)**: A command-line tool used to establish SSH connections.
* **PowerShell**: A command-line shell and scripting language for Windows.
* **Power User**: A user with above-average computer knowledge and skills.
* **Process Management**: The management of running programs and system resources.
* **Qwiklabs**: An online platform for cloud and IT training.
* **Remote Connection**: The ability to access and manage a computer from another location.
* **Remote Desktop Protocol (RDP)**: A protocol used to remotely access another computer.
* **Scalability**: A system's ability to adjust performance based on demand.
* **Shell**: A program that interprets commands and communicates with the operating system.
* **SSH (Secure Shell)**: A protocol used to securely connect to remote computers.
* **SSH Authentication Key**: A secure method of authenticating SSH connections.
* **SSH Client**: Software used to initiate an SSH connection.
* **SSH Server**: Software that accepts incoming SSH connections.
* **Standardization**: The use of common technical standards across systems and products.
* **Swap Space**: Disk space used as additional memory when RAM is full.
* **System**: Hardware and software components working together to perform tasks.
* **System Settings**: Configuration options for a computer, such as display and network settings.
* **Taskbar**: A bar that provides quick access to applications and system information.
* **Terminal**: A text-based application used to interact with a shell.
* **Time Slice**: A short amount of CPU time allocated to a process.
* **Ubuntu**: A popular Linux distribution.
* **Username**: A unique identifier for a user account.
* **User Space**: The part of the operating system where user applications run.
* **VirtualBox**: Software used to create and run virtual machines.
* **Virtual Machine (VM)**: A software-based computer that can run its own operating system.
* **Virtual Memory**: A combination of RAM and disk space used as additional memory.
* **VPN (Virtual Private Network)**: A secure connection to a private network over the Internet.
