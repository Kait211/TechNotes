- **ARPANET**: The earliest version of the Internet that we see today, created by the US government project DARPA in the 1960s.

- **Children's Online Privacy Protection Act (COPPA)**: Regulates the information we show to children under the age of 13.

- **Clients**: Devices that receive data from a server.

- **DARPA**: A US government project in the 1960s that went on to create the earliest version of the Internet that we see today.

- **Domain name**: A website name; the part of the URL following www.

- **Domain Name System (DNS)**: A global and highly distributed network service that resolves website names into IP addresses.

- **Ethernet cable**: A cable that physically connects a device to a network.

- **Fiber optic cable**: A cable containing tiny glass fibers that use pulses of light to transmit data.

- **Globalization**: The movement that enables governments, businesses, and organizations to communicate and integrate internationally.

- **Hubs**: Devices that serve as a central location through which data travels.

- **Internet**: A worldwide system of interconnected networks.

- **Internet Corporation for Assigned Names and Numbers (ICANN)**: The organization where website names are registered.

- **Internet of Things (IoT)**: The concept of connecting everyday devices to the internet in a smart way.

- **Internet Protocol version 4 (IPv4)**: A 32-bit address divided into four groups.

- **Internet Protocol version 6 (IPv6)**: A 128-bit address, providing far more addresses than IPv4.

- **Internet service provider (ISP)**: A company that provides internet access to consumers.

- **IP address**: A numerical label assigned to a device on a network that uses the Internet Protocol.

- **MAC address**: A globally unique 48-bit identifier assigned to a network interface.

- **Network**: An interconnection of computers.

- **Network Address Translation (NAT)**: A tool that allows many private IP addresses to share one public IP address.

- **Networking**: The process of managing, building, and designing networks.

- **Networking protocols**: A set of rules for transferring data across a network.

- **Network stack**: A collection of hardware or software that provides network infrastructure.

- **Router**: A device that forwards data between different networks.

- **Server logs**: Text files that record activities performed on a web server.

- **Servers**: Devices that provide data to clients.

- **Switches**: Devices that help data travel through a network.

- **Transmission Control Protocol (TCP)**: A protocol that ensures reliable delivery of information between networks.

- **Uniform Resource Locator (URL)**: A web address used to locate resources on the Internet.

- **WannaCry Attack**: A cyberattack that spread worldwide and infected hundreds of thousands of computers.

- **Wireless networking (Wi-Fi)**: Networks that use radio waves and antennas for communication.

- **World Wide Web (WWW)**: The information system that allows documents and resources to be accessed over the Internet.
